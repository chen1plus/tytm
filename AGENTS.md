# AGENTS.md

- Rust CLI crate, single binary `tytm`; source entrypoint is `src/main.rs`.
- Commands:
  - `add <theme_id>`: Installs a theme from `src/registry.json` by ID.
  - `list` (alias `ls`): Lists installed themes.
  - `remove <theme_id>` (alias `rm`): Uninstalls the theme and its variants.
  - `update`: Updates all installed themes to their latest version.
- Storage directories (themes folder `TYPORA_THEME`):
  - Debug builds: `./debug` (relative to current directory).
  - Release builds: OS-specific Typora themes directory:
    - macOS: `~/Library/Application Support/abnerworks.Typora/themes`
    - Windows: `%APPDATA%/Typora/themes`
    - Linux: `~/.local/share/typora/themes`
  - Manifest file: `TYPORA_THEME/tytm/installed.json`.
- Theme registry: Defined in `src/registry.json` and embedded at build time.
- Theme sources (`Source` enum):
  - `GhReleaseZip`: Download latest release zip from GitHub, extract specified files.
  - `Zip`: Download zip from URL, extract specified files.
  - `Git`: Git-based install (clones the repository, finds the base directory containing the theme files, and copies files).
- CI runs `cargo check` and `cargo test` only.
